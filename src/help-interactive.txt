Commands:
    /setmodel <INTERFACE:MODEL>: change the model on the fly
    /save <FILE>: save the chat session to a file
    /setfile [FILE]: set where to save the chat. Can also be used to remove it if called without arguments
    /exit: exit the chat, alternatively to ^D
    /fromfile <FILE>: use file's contents as the user prompt
    /status: show the current model, messages number, and autosave file
    /nvm: delete the last model response and user prompt pair
